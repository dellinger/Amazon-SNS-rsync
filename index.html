Amazon-sns-rsync

Rsync scripts to run a backup from source to destination on a daily / weekly basis. Uses Amazon SNS to notify on status.

Rsync backup scripts

The purpose of these scripts is to run a daily / weekly backup.

Amazon Simple Notification Service is used to abstract notifications from the script.

Amazon SNS uses the notion of "Topics" which can be defined in the Amazon Management Console (or through the command line interface). The Topic I'll be using is "rsync-daily" and "rsync-weekly".

Now, the script will send a message depending on the rsync completion status to the topic. I can subscribe my phone number or email address to get a text message or email address depending on what I would like.

The settings I currently have set is,

rsync-daily = Only send me a text message on ANY FAILURE.

rsync-weekly = Send me a detailed email on ANY completion status. If SUCCESS then send me a capacity of my source and a capacity of my destination. If FAILURE, send me the failure exception.
